# antigravity.google 官网粒子真实逻辑（从 main-M3OAQNLT.js 逆向提取）

## 整体架构
- Angular 应用，粒子在 `<app-root>` 内的 `morphingParticlesContainer` 渲染
- 类层级：`da`(组件) → `sI/J9`(scene) → `rI/K9`(particles)
- 用 GSAP (`ut`=gsap) 做 hoverProgress / pushProgress 缓动动画
- 依赖 Three.js（打包进 bundle），用 PerspectiveCamera(40°) + cameraZoom=3.5
- 鼠标通过 Raycaster 射线打到 raycastPlane 得到 intersectionPoint

## 1. 粒子分布：图片驱动的变密度泊松盘采样
- 使用 `poisson-disk-sampling` npm 库（VariableDensityPDS）
- shape: [500, 500]
- minDistance: 1, maxDistance: linearMap(density, 0, 300, 10, 50), tries: 20
- **distanceFunction 基于图片像素亮度**：`pixel = imageData.data[r]/255; return pixel*pixel*pixel`
  → 亮的区域粒子密，暗的区域稀疏，粒子形状由 icon 图片决定
- 采样在 Web Worker 里执行，返回 points + nearestPoints
- 点坐标缩放：存入 DataTexture 时 `*(1/250)`，所以 500x500 图 → [-1,1] 范围
- count = points.length，纹理 size=256 (length=65536)，超出部分留空

## 2. 数据纹理结构（GPGPU）
Float32Array(length*4)，每像素 RGBA = [pos.x, pos.y, scale, velocity]
- createDataTexturePosition: x=data[i*2]/250, y=data[i*2+1]/250, z=0(scale), w=0(velocity)
- 两张纹理：posTex(初始位置/refPos) + posNearestTex(最近点，hover 时目标)
- 两个 WebGLRenderTarget rt1/rt2 ping-pong，type=HalfFloat(Ka), RGBA, NearestFilter

## 3. SIM shader（核心物理，fragment）— 逐像素更新
```glsl
uniform sampler2D uPosition;   // 当前帧 [x,y,scale,velocity]
uniform sampler2D uPosRefs;    // 静息位置 refPos
uniform sampler2D uPosNearest; // 最近点位置（hover 目标）
uniform vec2 uMousePos;
uniform float uTime, uDeltaTime, uIsHovering;

void main() {
  vec2 uv = gl_FragCoord.xy / size;   // 256
  vec4 pFrame = texture2D(uPosition, uv);
  float scale = pFrame.z;
  float velocity = pFrame.w;
  vec2 refPos = texture2D(uPosRefs, uv).xy;
  vec2 nearestPos = texture2D(uPosNearest, uv).xy;
  float seed = hash(uv).x, seed2 = hash(uv).y;

  float time = uTime * .5;
  float lifeEnd = 3. + sin(seed2*100.)*1.;        // 生命周期 2~4s
  float lifeTime = mod((seed*100.) + time, lifeEnd);

  vec2 pos = pFrame.xy;
  float distRadius = 0.15;
  // hover 时目标在 refPos 和 nearestPos 间插值（鼠标推开粒子到最近邻）
  vec2 targetPos = mix(refPos, nearestPos, uIsHovering * uIsHovering);
  vec2 direction = normalize(targetPos - pos) * .01;
  float dist = length(targetPos - pos);
  float distStrength = smoothstep(distRadius, 0., dist);  // 越近力越大
  if (dist > 0.005) pos += direction * distStrength;      // 向心引力拉回

  // 生命重置：到时间重置位置
  if (lifeTime < .01) { pos = refPos; scale = 0.; }

  // scale 渐入渐出（呼吸感）+ hover 放大
  float targetScale = smoothstep(.01, 0.5, lifeTime) - smoothstep(0.5, 1., lifeTime/lifeEnd);
  targetScale += smoothstep(0.1, 0., smoothstep(0.001, .1, dist)) * 1.5 * uIsHovering;
  scale += (targetScale - scale) * .1;   // 平滑过渡

  // velocity = 接近程度 * hover
  velocity = smoothstep(distRadius, .001, dist) * uIsHovering;

  gl_FragColor = vec4(pFrame.xy + diff, scale, velocity);
}
```
**关键：官网没有 curl noise / 惯性 / 自由漂浮！** 它是：
- 向心引力拉回静息位置（distStrength）
- 生命周期重生（fade in/out 产生闪烁呼吸）
- hover 时目标切到 nearestPos（粒子被鼠标"推开"到邻位点）

## 4. RENDER shader（顶点）
```glsl
attribute vec4 seeds;
uniform sampler2D uPosition;
uniform float uTime, uParticleScale, uPixelRatio, uIsHovering, uPulseProgress;

void main() {
  vec4 pos = texture2D(uPosition, uv);
  vSeeds = seeds;

  // 两层 simplex 噪声做漂浮抖动
  float noiseX = snoise(vec3(pos.xy*10., uTime*.2+100.));
  float noiseY = snoise(vec3(pos.xy*10., uTime*.2));
  float noiseX2 = snoise(vec3(pos.xy*.5, uTime*.15+45.));
  float noiseY2 = snoise(vec3(pos.xy*.5, uTime*.15+87.));

  // 径向脉冲波（pushProgress 从中心向外传播）
  float cDist = length(pos.xy);
  float progress = uPulseProgress;
  float t = smoothstep(progress-.25, progress, cDist) - smoothstep(progress, progress+.25, cDist);
  t *= smoothstep(1., .0, cDist);
  pos.xy *= 1. + (t * .02);   // 脉冲处微扩张

  // hover 时近鼠标的粒子抖动更大
  float dist = smoothstep(0., 0.9, pos.w);  // pos.w=velocity
  dist = mix(0., dist, uIsHovering);
  pos.y += noiseY * .005 * dist;
  pos.x += noiseX * .005 * dist;
  pos.y += noiseY2 * .02;     // 全局漂浮
  pos.x += noiseX2 * .02;

  vVelocity = pos.w; vScale = pos.z; vLocalPos = pos.xy;
  gl_Position = projectionMatrix * modelViewMatrix * vec4(pos.xy, 0., 1.);

  // 点大小：随 scale + colorScheme 基础值
  float minScale = .25 + float(uColorScheme)*.75;
  gl_PointSize = ((vScale*7.)*(uPixelRatio*0.5)*uParticleScale) + (minScale*uPixelRatio);
}
```

## 5. RENDER shader（片段）— 圆角矩形 + disc
```glsl
float sdRoundBox(vec2 p, vec2 b, vec4 r) {
  r.xy = (p.x>0.)?r.xy:r.zw; r.x = (p.y>0.)?r.x:r.y;
  vec2 q = abs(p)-b+r.x;
  return min(max(q.x,q.y),0.) + length(max(q,0.)) - r.x;
}

void main() {
  float angle = atan(vLocalPos.y-uMousePos.y, vLocalPos.x-uMousePos.x);  // 指向鼠标角度（未使用绘制，仅计算）
  vec2 uv = gl_PointCoord.xy - 0.5;  uv.y *= -1.;
  float dist = length(uv);

  // 颜色：按 velocity 三段渐变 color1→color2→color3
  float h = 0.8;
  vec3 col = mix(mix(c1,c2,vel/h), mix(c2,c3,(vel-h)/(1.-h)), step(h,vel));

  // 圆角矩形（注意 b=vec2(0.5,0.2) → 宽长比，r=.25 圆角）
  float rounded = sdRoundBox(uv, vec2(0.5,0.2), vec4(.25));
  rounded = smoothstep(.1, 0., rounded);
  float disc = smoothstep(.5, .45, length(uv));   // 圆盘

  float a = uAlpha * disc * smoothstep(0.1, 0.2, vScale);  // 用 disc 作 alpha
  if (a < 0.01) discard;
  gl_FragColor = vec4(col, clamp(a, 0., 1.));
}
```
**重要发现：官网粒子形 = 圆角矩形(sdRoundBox b=0.5,0.2 r=0.25)，但最终 alpha 用的是 disc(圆盘)！**
→ 实际渲染是**圆点**，sdRoundBox 算了但没用于 alpha。所以官网是圆点星云。
**这就是你要改的：把 disc 换成 capsule/圆角矩形，让粒子变成长条胶囊。**

## 6. update() 每帧逻辑
```
mousePos = intersectionPoint * 0.175    // 缩小鼠标影响范围
particleScale = canvas.width / pixelRatio / 2000 * particlesScale
uRingRadius = 0.175 + sin(time)*0.03 + cos(time*3)*0.02   // 环呼吸
simMaterial: uPosition(rt1或posTex), uTime, uDeltaTime, uMousePos, uIsHovering(hoverProgress)
→ render simScene 到 rt2
renderMaterial: uPosition(rt2), uTime, uMousePos, uParticleScale, uIsHovering, uPulseProgress(pushProgress)
→ render 主场景
postRender: 交换 rt1/rt2, everRendered=true
```

## 7. 交互缓动（GSAP）
- onHoverStart: hoverProgress→1 (0.5s power3.out), pushProgress 0→1 (2s power2.out, pulse波)
- onHoverEnd: hoverProgress→0, pushProgress 0→1 (再触发一次脉冲)
- setPointsTextureFromIndex: 切换 nearestPoints 纹理时也触发脉冲

## 总结：官网底层逻辑 = 向心引力 + 生命周期呼吸 + hover推到邻点 + 脉冲波
**不是流体/curl/惯性。** 你要的水母改造点：
1. 粒子形：disc → capsule（sdRoundBox 改长条 + 用作 alpha）
2. 朝向：让长条指向中心（用 vLocalPos 与中心 atan 旋转）
3. 整体意象：俯视水母（无实体，只靠粒子轮廓）
